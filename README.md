# Silicon Logic: NAND to Tetris

I am building a 16-bit computer system from first principles. This repository is a record of my exploration into computer architecture, moving from Boolean logic to a functional CPU.

### 🧩 My Philosophy
I believe in understanding systems by dismantling abstractions. My goal is to master the logic that sits between the physical electron and high-level code.

### 🏗 Progress
- [x] **Boolean Logic:** Designing basic gates (Mux, DMux).
- [x] **Boolean Arithmetic:** Building a 16-bit ALU from scratch.
- [ ] **Sequential Logic:** Implementing Registers and RAM.
- [ ] **Architecture:** Connecting the CPU and Memory.

### 🔬 Exploration
I am focused on gate optimization, minimizing chip counts, and documenting the "why" behind every connection. 

**I welcome technical suggestions or discussions on low-level engineering.**
